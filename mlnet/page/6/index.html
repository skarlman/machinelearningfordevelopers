INTRODUCTION TO ML.NET (DevReach 2018)
ML.NET is aimed at providing a first class experience for Machine Learning in .NET. Using ML.NET, …
ML.NET is aimed at providing a first class experience for Machine Learning in .NET. Using ML.NET, .NET developers can develop and infuse custom Artificial Intelligence into existing .NET apps through a code-driven and UI driven approaches. ML.NET has been used extensively within Microsoft by Windows, Azure, SQL & Bing technologies for a decade and now all the capabilities are available to all .NET developers. Come be a part of the journey as ML.NET is built out in the open with developer collaboration. This session will go into scenarios that ML.NET enables, along with vision of architecting apps with ML.NET.
